Quick Notes

This admin panel is backed by a real SQLite database via Django. Every change here — hero text, products, gallery, review moderation — is written to the database immediately and is visible to every visitor and every admin, on any device. Orders and review submissions from the storefront are also saved here automatically and notify you on WhatsApp. Use Preview Site in the sidebar to see the live storefront update in real time, right here in the dashboard.

About this admin

This dashboard reads and writes a real SQLite database via Django models — every save here is permanent and shared across all admins and devices. Log-ins are real Django user accounts; create more admins with python manage.py createsuperuser. Django's own built-in admin is also available at /admin/ for bulk edits.
